# Catstronaut HUD

An immersive scroll-scrubbed astronaut cat microsite built as a static HTML/CSS/JS project.

## Local Preview

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173/`.

## Structure

- `index.html` - page markup and social metadata.
- `styles.css` - responsive layout, glass panels, and 3D typography.
- `script.js` - scroll-linked frame sequence and reveal behavior.
- `frames/` - extracted frame sequence from the source video.
- `a52d273c-b1c8-477b-94d6-5cb4f8979f6b.mp4` - original source video.
